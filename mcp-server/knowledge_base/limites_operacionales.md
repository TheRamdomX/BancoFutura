# Límites operacionales — BancoFutura

## Transferencias
- Máximo por operación: $5.000.000 CLP.
- Máximo diario acumulado: $10.000.000 CLP.

## Giros en cajero
- Máximo por giro: $200.000 CLP.
- Máximo diario en cajeros: $400.000 CLP.

## Compras con tarjeta
- Débito: $500.000 CLP diarios por defecto.
- Crédito: hasta el cupo aprobado, máximo $1.000.000 CLP diarios.

## Apertura de productos
- El cliente puede tener hasta 2 cuentas corrientes y 2 de ahorro.
- Cada cuenta puede tener hasta 3 tarjetas asociadas.

## Comisiones
- Transferencias entre cuentas BancoFutura: sin costo.
- Transferencias interbancarias: sin costo para clientes con plan vigente.
- Mantención de cuenta corriente: $0 con plan digital.
